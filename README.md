# 🌱 Agro Sustentável

Projeto desenvolvido para apresentar informações, conteúdos e recursos relacionados à **agricultura sustentável**, promovendo educação, conscientização ambiental e práticas agrícolas responsáveis através de uma plataforma web moderna e responsiva.

## ⚙️ Como Executar Localmente

1. Clone o repositório:

```bash
git clone https://github.com/davialvessantos22-cyber/daviagrinho.git
```

2. Acesse a pasta do projeto:

```bash
cd daviagrinho
```

3. Abra o arquivo `index.html` no navegador.

Ou, se utilizar Node.js:

```bash
npm install
npm start
```

## 🎯 Objetivo

Este projeto foi desenvolvido com o objetivo de incentivar o aprendizado sobre **agricultura sustentável**, práticas ecológicas no campo e a importância da conexão entre meio rural e preservação ambiental.
